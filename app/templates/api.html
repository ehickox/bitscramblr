Bitscrambl your applications!

Want to provide anonymous Bitcoin transactions for your users? Now you can with Bitscramblr's RESTful API!

It's as simple as sending a GET request to http://www.bitscramblr.com/api/receive

Include parameters:
destination
amount

We'll generate a new Bitcoin address and return it in our response. From there, once the address we provide is filled, we'll send Bitcoin to the destination you provide.
Our response should look something like this:
{ 'address': 'XXX', 'amount': XXX, 'destination: 'YYY' }
